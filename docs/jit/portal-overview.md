# JIT Access portal overview

Use the JIT portal to review license capacity, configure roles and assignments, activate eligible access, and monitor active sessions.

## Dashboard

![JIT dashboard overview](./screenshots/dashboard-overview.png)

| Area | Purpose |
| --- | --- |
| **JIT Status** | Shows license state, assigned identities, signed-in user, and remaining licensed users. |
| **Start Work** | Opens the JIT workspace, Settings, and licensed-user actions. |

## JIT Access overview

![JIT Access overview](./screenshots/jit-access-overview.png)

The overview summarizes configured roles, assignments, and active sessions. Use it before and after configuration changes.

## Roles

![JIT roles list](./screenshots/roles-list.png)

**Roles** shows the role name, enabled state, allowed assignment types, duration limits, OTP requirement, and mapped Active Directory groups. A role alone does not activate access.

## Assignments

![JIT assignments list](./screenshots/assignments-list.png)

**Assignments** connects users to roles through Manual, Scheduled, or Eligible OTP access.

## Active Sessions

![JIT active sessions](./screenshots/active-sessions.png)

**Active Sessions** shows current access, expiration, and available extend or revoke actions.

## Settings

![JIT settings overview](./screenshots/settings-overview.png)

**Settings** manages licensed users, JIT RBAC assignments, product behavior, notifications, SMTP, and portal session policy.
